# RAG for Patient Information

This repository contains materials supporting a research letter evaluating Retrieval Augmented Generation (RAG) for patient facing health information.

It includes test questions, prompt templates, evaluation outputs, and supporting artefacts (for example the RAG architecture and data flow diagram), alongside consolidated Python requirements to support reproducibility.

## What is in this repo

Key folders and files (as currently structured):

- `Test_Data/`: test items used for the evaluation
- `LFTK_Result_Data/`: outputs relating to automated readability and text feature analyses
- `QAMAI_Results_Experts.xlsx`: expert rating results
- `QAMAI_LLM_Judge_Results.xlsx`: LLM judge results
- `LLM_Judge_Base_Prompt.docx`: base prompt used for the judge setup
- `Parameters and prompt templates.docx`: prompt and parameter settings used to generate and evaluate outputs
- `RAG Architecture and Data Flow.docx`: system level diagram and description
